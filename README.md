# Introduction
本项目为jupyter的magic，目的是方便在jupyter中直接运行python代码。

有关jupyter magic的介绍，参见：
- [Built-in magic commands](https://ipython.readthedocs.io/en/stable/interactive/magics.html)

# Install

首先构建并安装该package：

```bash
sh install.sh
```
